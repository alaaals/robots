Welcome to my Robot game!


Detailes on game play:
This game consists of an infinite number of robots (some with different physical features) flowing from the middle left of the screen onto a grib of arrows. By clicking the arrows, the user can control the direction that the robot(s) are going in if the robot(s) happen to be in the vicinity of an arrow. Scattered on a random selection of arrows are coins that the user should maneuver the robots to collect; each coin increases the user's score by one. Finally, after collecting the coins the user should direct the robots to the final door found on the bottom left of the screen, which will then count the number of robots that pass through it.

Creation of game
This game consists mainly of two classes, an arrow class and a robot class. From there, I created two arrays, one from each class, and allowed them to interact in specific ways. If a robot is on a "left" arrow it moves left, if it's on a "right" arrow it moves right, and so on with up and down. The arrows detect if the mouse is clicking on them, and if it is, then with each click it moves from right to down to left to up, in that rotational order. Visual aspects such as the flickering yellow circles on the robots were added as directional indicators. Robot can have different eye styles, limbs, and colors. Arrows are loaded graphics that change their graaphics once a mouse clicks on them.
